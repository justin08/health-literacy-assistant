# Cyclobenzaprine
Type: Medication / Drug Information

Cyclobenzaprine is used  to treat  muscle spasms. Cyclobenzaprine is in a class of medications called skeletal muscle relaxants. It works by acting in the brain and nervous system to allow the muscles to relax.
