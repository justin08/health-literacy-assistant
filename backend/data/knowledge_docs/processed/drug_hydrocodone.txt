# Hydrocodone
Type: Medication / Drug Information

Hydrocodone is used to relieve severe pain. Hydrocodone is in a class of medications called opiate (narcotic) analgesics. It works by changing the way the brain and nervous system respond to pain. This monograph only includes information about the use of hydrocodone alone. If you are taking a hydrocodone combination product, be sure to read information about all the ingredients in the hydrocodone-combination monograph and ask your doctor or pharmacist for more information.
